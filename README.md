# Github 组织用户备注

一个 Chrome Manifest V3 插件，用于给 GitHub 用户打备注，并在页面中展示备注名。

支持中英文界面（默认英文，可在弹窗右上角一键切换）。

## 已支持场景

- 组织成员列表（`/orgs/<org>/people`）
- 组织项目管理里涉及用户展示的位置（只要是用户链接）
- 组织设置中负责人/管理相关的用户操作列表（只要是用户链接）

实现方式是内容脚本监听页面 DOM 变化并动态补充备注，因此对 GitHub 的异步加载页面也能工作。

## 安装（开发者模式）

1. 打开 Chrome `chrome://extensions`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录：
   - `/Users/apple/Dev/JXCQ/github-remark`

## 使用方法

1. 打开插件弹窗
2. 输入 GitHub 用户名和备注名并保存
3. 回到 GitHub 页面，用户名右侧会出现备注徽标
4. 无备注时会显示“+备注”按钮，点击可快速新增
5. 点击页面上的备注徽标可快速编辑
6. 在插件弹窗可导出/导入 JSON 备注数据（导入为合并模式）

备注存储在 `chrome.storage.sync`，同一个 Chrome 账号可同步。
